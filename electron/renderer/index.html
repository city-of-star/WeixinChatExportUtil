欢迎使用微迹

在本地电脑导出微信 PC 版聊天记录，数据不会离开你的设备。

隐私说明

  • 本工具完全在本地运行，不会上传、同步或分享你的聊天记录
  • 所有读写操作仅针对你电脑上的微信数据目录和导出文件夹
  • 我们不会收集任何个人数据或聊天内容

使用流程

  1. 选择账号 — 定位微信数据目录，选择要导出的账号
  2. 扫描会话 — 首次可能需要几分钟；若需解密，工具会暂时关闭微信并自动重新打开
  3. 登录微信 — 微信弹出后，请在窗口中点击「登录」（通常无需重新扫码）
  4. 选择会话并导出 — 勾选要导出的聊天，选择保存位置和格式

导出后你会得到

导出文件夹/ ├── index.html — 用浏览器打开，浏览所有会话（推荐） ├── conversations.json — 会话索引与元数据 ├── contacts.json — 联系人昵称对照表 ├── messages.csv — 全部消息汇总（勾选 CSV 时） └── chats/ — 每个会话单独的 HTML / JSON / TXT 文件
免责声明

本工具为第三方开源工具，与腾讯微信无任何关联。使用前请确认：

  • 你导出的聊天记录仅供个人备份与查阅，请遵守相关法律法规
  • 请勿将他人隐私内容用于非法用途或未经授权的传播
  • 解密过程需要读取微信本地数据库，请确保你有权访问该账号数据
  • 因使用本工具产生的任何后果，由使用者自行承担
