欢迎使用微迹

在本地电脑导出微信 PC 版聊天记录，数据不会离开你的设备。

隐私说明

  • 本工具完全在本地运行，不会上传、同步或分享你的聊天记录
  • 所有读写操作仅针对你电脑上的微信数据目录和导出文件夹
  • 我们不会收集任何个人数据或聊天内容

工作原理

  • 读取本地数据 — 从 xwechat_files 账号目录读取加密的 SQLite 数据库
  • 获取解密密钥 — 首次使用时在本机捕获密钥；完成后密钥与解密结果保存在本地，无需每次重复
  • 扫描与导出 — 解密后统计会话列表，再按你勾选的格式写入导出文件夹
不同微信版本如何获取密钥？

工具会自动识别微信版本并选用对应方式，无需手动选择。

  • 4.1.9 及更早 — 登录后密钥在进程内存中，扫描已登录的微信进程定位
  • 4.1.10 及之后 — 密钥仅在登录瞬间出现，通过内置 DLL Hook 拦截；扫描时会提示何时点击「登录」

两种方式均在本机完成；解密后的数据库保存在账号目录的 db_storage_decrypted 中，由你完全控制。

导出后你会得到

实际文件取决于你勾选的导出格式（可多选）

导出文件夹/ ├── index.html — 勾选 HTML 时，浏览器浏览所有会话（推荐） ├── conversations.json — 勾选 JSON 时，会话索引与元数据 ├── contacts.json — 勾选 JSON 时，联系人昵称对照表 ├── messages.csv — 勾选 CSV 时，全部消息汇总表 └── chats/ — 各会话单独的 HTML / JSON / TXT 文件

    1

    导出总览

    确认保存位置与格式，然后开始导出。

    预计时间怎么理解？
    • 根据当前勾选的会话、消息数量、导出格式和本机运算速度估算；Lv.1–10 代表电脑处理快慢，数字越高越快
    • 显示的是参考区间，不是精确倒计时；磁盘、首次解密等因素可能让实际时间有所出入
    • 勾选语音转写后会明显变长；语音若此前已转过，重复导出会更快
    • 开始导出后，进度条显示总进度（含读取、转写、写入），旁侧会显示剩余时间，通常比这里的预估更准确
    2

    保存位置

    保存在哪里比较合适？

    建议选择空文件夹。导出完成后会自动生成 index.html、conversations.json 及按会话拆分的文件。

    3

    导出格式

    各格式有什么用途？
    • HTML — 双击 index.html 用浏览器浏览，最直观
    • JSON — 结构化数据，便于程序处理
    • TXT — 纯文本，每个会话单独文件
    • CSV — 表格汇总，适合 Excel 分析
    4

    语音转文字

    启用后有什么影响？
    • 每条语音约需数秒识别，导出时间可能明显变长；语音较多时建议先导出少量会话试跑
    • 工具已内置语音识别模型，全程本地识别、无需联网
    • 转写结果会缓存在微信账号目录，重复导出更快
    • 识别结果会写入 JSON / TXT / HTML / CSV，不保存原始语音文件
    • 识别准确率受口音、背景噪音影响，仅供参考
